# Ingestion Module

Owner: Diptangshu

## Purpose
Convert syllabus documents (PDF) into structured JSON matching the locked hierarchical model.

## Responsibilities
- Extract raw text from PDF
- Detect units and topics
- Output structured JSON
- Do NOT modify data model

## Not Allowed
- No priority logic here
- No ML implementation
- No frontend code

## Expected Output
Structured JSON compatible with planner engine.
